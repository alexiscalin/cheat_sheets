TMUX CHEAT SHEET

Ultimate Multi-terminal Management
Default Prefix
Ctrl+b
Press prefix, then key.
🚀 Quick Start
  1. Connect: ssh user@server
  2. Launch: tmux new -s work
  3. Split/Run: Create panes, execute commands.
  4. Disconnect: Press Ctrl+b then d
  5. Resume: tmux a -t work
Install: sudo apt install tmux (Debian/Ubuntu)
💻 Session CLI Management
CommandDescription
tmux new -s <n>New session named <n>
tmux new -s <n> -dCreate in background
tmux lsList active sessions
tmux a -t <n>Attach to named session
tmux aAttach to last active
tmux kill-session -t <n>Kill named session
tmux kill-serverKill all sessions/server
tmux rename-session <n>Rename current session
🔗 Session Keybindings
Inside active session:
ShortcutAction
Ctrl+b dDetach session (keep active)
Ctrl+b sInteractive session list
Ctrl+b $Rename current session
Ctrl+b (Switch to previous session
Ctrl+b )Switch to next session
Your processes survive SSH disconnections.

◫ Panes (Splits)

📄 Copy Mode (Scrollback)
ShortcutAction
Ctrl+b [Enter scroll/copy mode
SpaceBegin selecting text
ArrowsNavigate cursor selection
PgUp / PgDnScroll terminal history
EnterCopy select text to buffer
Ctrl+b ]Paste buffer contents
q / EscExit copy mode
🔍 Search (In Copy Mode)
ShortcutAction
/Search forward
?Search backward
nGo to next match
NGo to previous match

📐 Pane Resizing
Shortcut/CmdAction
Ctrl+b Arrows (Hold)Resize active pane 1 cell
Ctrl+b Alt+ArrowsResize active pane 5 cells
resize-pane -L <n>Grow pane Left by <n>
resize-pane -R <n>Grow pane Right by <n>
resize-pane -U <n>Grow pane Up by <n>
resize-pane -D <n>Grow pane Down by <n>
